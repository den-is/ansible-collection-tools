# zed - text editor

## Links
- https://github.com/zed-industries/zed

## Instructions
This ansible role follows official installation for linux hosts from [here](https://zed.dev/docs/linux#standard-installation).  
It is is working with `~/.local` only for a given user. (i.e should not be running with sudo/become)  
It is recommended not to alter zed download/extraction/deploy paths in this role and keep them as is.
