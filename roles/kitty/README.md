# Kitty terminal emulator

## Links
- https://github.com/kovidgoyal/kitty
- https://sw.kovidgoyal.net/kitty/

## Instructions
This ansible role follows official installation for linux hosts from [here](https://sw.kovidgoyal.net/kitty/binary/).  
It is is working with `~/.local` only for a given user. (i.e should not be running with sudo/become)  
It is recommended not to alter kitty download/extraction/deploy paths in this role and keep them as is.
